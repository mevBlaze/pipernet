## The Atom — the DOT

A **DOT** is a self-sovereign state container with four properties:

1. **Address** — your DOT, one universal address that replaces email/phone/handle.
2. **Membrane** — the permission layer that decides what to reveal to whom.
3. **Contents** — state, timeline, signature, whatever's inside.
4. **Format independence** — described at the level of pure information so any sufficiently intelligent civilization could reconstruct it.

Wire format: 153 bytes. Ed25519 keypair as identity. SHA-256 chain integrity. Content-addressed. Append-only.

Reference SDK: `js/dot.mjs` — zero dependencies, Web Crypto API, Node 18+.

## The architecture

```
HUMAN LAYER     — Clean. Visual. Private. Calm. Designed for presence and trust.
       │
   THE SPINE    ← the real innovation: identity, addressing, verification, routing
       │
AGENT LAYER     — No UI. Pure API. High-frequency. Structured. Reputation + trust.
```

State, not message. Every message is a delta — a diff, like git. We don't store messages. We store states. What you see in the conversation is a timeline of state changes — two minds editing a living document just by existing in it.

The **handshake** is the atom of the protocol stack. The **behavioral signature** is the foundational identity primitive — a continuous living signature that proves any message (human or agent) is a coherent expression of an actual self. It lives at the edge. It never leaves your device. **Proof of self.**
